# AI Resume Job Matching System

Resume screening and job matching system using semantic embeddings and skill-level analysis.

## Features
- Resume PDF parsing
- Semantic similarity matching
- Keyword-based ATS scoring
- Explainable results (matched, missing, extra skills)
- Works for any job role

## Tech Stack
- Python
- Streamlit
- Sentence Transformers
- Scikit-learn

## Run the Project
```bash
pip install -r requirements.txt
streamlit run jobmatching.py
